# rust-radio-sx128x

A rust driver (and command line utility) for the [Semtech SX128x](https://www.semtech.com/products/wireless-rf/24-ghz-transceivers/sx1280) 2.4GHz ISM band radio IC.


## Status

WIP. Basic LoRa functionality working.
